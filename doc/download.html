Download Couchbeam

This is a manual for installing couchbeam and its dependencies.

Requirements

  • Erlang/OTP R12-B or newer (compiler to build, kernel,stdlib,ssl,crypto to run)
  • GNU Make (might actually build with some other make as well)

Get the sources

You could download latest sources from Github dowload pageor fetch the sources with git. Therefore we have to install git and then run : 

    git clone git://github.com/benoitc/couchbeam.git
An hg mirror is also available on bitbucket.

installation

To build the application simply run 'make'. This should build .beam, .app files and documentation.

To run tests run 'make test'.
To generate doc, run 'make docs'.
